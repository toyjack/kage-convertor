# Kage-Convertor

A convertor for Kage data.

## Prepare

You should download GlyphWiki's dump file by yourself from [this link](http://glyphwiki.org/dump.tar.gz). Then put the `dump_newest_only.txt` into the root of this directory.

## Install & Use

```bash
npm install
npm run build
node ./dist/index.js
```
